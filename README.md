# Dancing Mad (Ultimate) / "UMAD" — FFXIV Strategy Guide (English)

A static, ad-free **English translation** of the FINAL FANTASY XIV high-end raid guide for
**絶妖星乱舞 / Dancing Mad (Ultimate)** ("UMAD"). Phase-by-phase mechanics, diagrams, timeline
tables and party-finder macros — the full guide, nothing else.

👉 **Live site:** https://bigcatclaw.github.io/umad-guide/

## What this is

- A single static page (`index.html`) plus its diagram images (`assets/img/`) and stylesheets (`assets/css/`).
- No JavaScript, no ads, no trackers, no comment sections, no streams.
- All collapsible phase sections are expanded by default.
